Some of my projects

makemake

Makemake is a powerful command-line interface (CLI) program built with Rust, designed to simplify the process of creating and managing project templates. Whether you're a seasoned developer or just starting out, makemake is your go-to tool for boosting productivity and maintaining consistency across your projects. With makemake, you can effortlessly generate customized project templates that suit your specific needs. Say goodbye to repetitive setup tasks and hello to streamlined workflows. Simply define your desired project structure, including files, directories, and any other necessary assets, and let makemake do the heavy lifting for you.
GitHub AUR package (-S makemake)

makemake-templates

makemake-templates

Makemake-templates is a collection of some useful templates for the makemake. It includes templates for C, C++, LaTeX and more. The project also includes a script so you can esily install all the templates with just one simple command.
